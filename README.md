## What Is This?

This is my living document — a personal technology radar based on 21+ years of experience and 70+ delivered eCommerce projects.

## Adopt (I actively recommend these)

| Technology | Why | Ideal For |
|------------|-----|-----------|
| WooCommerce | True open-source flexibility | Small and Mid size projects, Quick deploy |
| Medusa.js | True open-source flexibility | Complex, custom needs |
| Shopify Headless | Speed + reliability | Brands needing UX control |
| Cloudflare | Security + CDN + DNS | Every project |
| AWS (ECS, RDS, Lambda) | Mature, scalable | Enterprise-grade needs |

## Hold / Assess (Proceed with caution)

| Technology | Why Cautious |
|------------|---------------|
| Magento | High TCO, complex maintenance |
| Wordpress unmaintained plugins | Security risks |

## Evaluate (On my radar)

- Saleor – GraphQL-first, Python backend
- AI Agents for customer support
- Medusa v2 – Modular architecture
